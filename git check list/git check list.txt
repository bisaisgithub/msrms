Note: always close IDE when working in github

First time steps

1. git clone repoURL
2. git checkout -b branchname
3. open the project in IDE and start coding and close IDE when done
4. git status
5. git add
6. git commit -m "notes/comment"
7. git push origin branchname
8. pull request github
9. delete branch in github and local


2nd time and so on
1. verify you are in master, else checkout
2. git pull origin master
3. git check -b branchname
4. open the project in IDE and start coding and close IDE when done
5. git status
6. git add
7. git commit -m "notes/comment"
8. git push origin branchname
9. pull request github
10. delete branch in github and local
